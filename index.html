About Smart Campus Placement & Career Portal

The Smart Campus Placement & Career Portal is a cloud-based placement management system developed using AWS services. This portal enables students, companies, and administrators to manage placement activities efficiently.

Features:

  • Student Registration
  • Company Registration
  • Resume Upload using Amazon S3
  • Placement Management
  • Admin Dashboard
  • Database Management using Amazon RDS
  • Cloud Monitoring using Amazon CloudWatch
